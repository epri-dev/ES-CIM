BacterialCountAnalog

Analogs

This is a measurement of the bacteria level present in a specific volume. If the bacteria type is known, the name field inherited from IdentifiedObject should be used, e.g. if measuring the bacteria count of plankton, then name = "Planktonic". If the name field is left empty or is not used, it is implied to be the total bacteria count.

ConcentrationAnalog

Analogs

This is a measurement of the concentration of an element, compound, or isotope in a specific volume. If the element, compound, or isotope is known, the name field inherited from IdentifiedObject should be used, e.g. if measuring the concentration of iron, then name = "Fe". If the name field is left empty or is not used, it is implied to be the total concentration from all constituents.
